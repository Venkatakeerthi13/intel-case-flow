# AI Powered Customer Support Ticketing System

## Description
A Salesforce-style AI-powered telecom customer support ticketing system developed using Lovable.

## Features
- Support Dashboard
- Case Management
- Accounts & Contacts
- Categories
- SLA Policies
- AI Feedback
- Reports & Analytics

## Technologies Used
- React
- TypeScript
- Tailwind CSS
- Supabase
- Lovable
